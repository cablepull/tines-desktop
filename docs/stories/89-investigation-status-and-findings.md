# User Story 89: Investigation Status And Findings

**As a** user conducting a forensic investigation,  
**I want** to record the status and my findings,  
**so that** the saved investigation reflects more than transient UI state.

## Acceptance Criteria
- An investigation includes a local status such as `Open`, `Needs Review`, `Resolved`, or `Archived`.
- An investigation includes a short summary.
- An investigation includes structured findings or conclusion notes.
- Reopening an investigation restores those fields.
