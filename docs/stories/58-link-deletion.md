# User Story #58: Link Deletion
**As a** Tines Automation Engineer,
**I want to** break existing connections by clicking a "Break Link" UI on the SVG path,
**so that** I can refactor my story structure without leaving the visual editor.

### Acceptance Criteria
- A clickable "×" icon appears on hover over SVG connection paths.
- Clicking the "×" icon removes the source ID from the target node.
- Real-time refresh of the visual graph after deletion.
