# User Story #64: Clean Logic Separation (Local vs Server)
**As a** Tines IDE User,
**I want** the UI to clearly distinguish between local "Safety Lock" (Client browsing safety) and server "Story Lock" (API persistence governance),
**So that** I don't get confused about why my edits are blocked.

### Acceptance Criteria
- Labeling differentiates "Safety Lock" (Local) and "Server Lock" (API).
- Visual states (colors/icons) help identify which tier of protection is active.
