# User Story 92: Investigation Browser

**As a** user with many saved investigations,  
**I want** a clearer investigations browser,  
**so that** I can locate relevant saved work quickly.

## Acceptance Criteria
- The dedicated Investigations page surfaces updated timestamps and key saved context.
- The browser can distinguish records by story, status, and saved run.
- The browser allows reopening an investigation into its saved story context.
- The UI remains local-first and does not depend on Tines-side persistence.
