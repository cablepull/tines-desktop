# Node Typology Customization

## Scenario
As an automation operator, complex graphs look like giant homogenous webs. I want Entry-point triggers (e.g. Schedule/Webhook agents) immediately distinguished via specific highlight colors so I visually identify exactly where my flows launch!

## Acceptance Criteria
- React render loops analyze `action.type`.
- Standard logic endpoints resolve to traditional purple/blue CSS visual classes.
- Triggers (`WebhookAgent`, `ScheduleAgent`) evaluate dynamically to Green badging, clearly indicating logic launch points visually.
