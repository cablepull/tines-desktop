# User Story 68: Story Ownership Context (Visual Badges)

## 📋 Story Header
- **ID**: 68
- **Title**: Story Ownership Context (Visual Badges)
- **Priority**: Medium
- **Status**: Completed (Phase 35)

## 👤 User Persona
As a power user managing dozens of stories, I want a visual indicator of story ownership on the dashboard so that I can quickly identify which flows are private and which are shared.

## 🎯 Acceptance Criteria
1. Each story card must display a unique icon (👤 for Personal, 👥 for Team).
2. The story card must display the actual Team Name instead of a raw numeric ID.
3. Visual indicators must be consistent with the selected workspace context.

## 📖 Description
Enhance story card metadata with visual badges and resolved names to provide immediate context of ownership and location within the Tines ecosystem.
