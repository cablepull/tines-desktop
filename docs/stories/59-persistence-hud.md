# User Story #59: Persistence HUD
**As a** Tines Automation Engineer,
**I want to** see a "Synced" or "Syncing" status indicator on the canvas,
**so that** I have confidence that my topology changes are being saved permanently to the Tines Cloud.

### Acceptance Criteria
- HUD displays real-time sync status: "Synced", "Syncing...", or "⚠️ Error".
- Shows "Last Saved" timestamp for user reassurance.
- Animates during active API writes.
