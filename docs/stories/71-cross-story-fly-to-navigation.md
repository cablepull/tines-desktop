# Story 71: Cross-Story "Fly-To" Navigation

## Scenario
As a developer, I found a specific HTTP Agent in the Global Actions list and want to edit its options.

## Acceptance Criteria
- Clicking the action row in the browser navigates to the Story Canvas.
- The canvas automatically pans and zooms to center precisely on that agent node.
- The node is temporarily highlighted with a ring to confirm the focus.
