# Board Comments / Annotations on Canvas

## Scenario
As a Security Engineer collaborating on automation workflows, I need to place sticky-note annotations on the canvas board to document decisions, flag risks, or leave instructions for teammates — just like the comment feature on the Tines cloud board.

## Acceptance Criteria
- A 📝 button in the canvas HUD adds a sticky note centered in the current viewport.
- Notes render as yellow cards on the canvas alongside action nodes.
- Notes are draggable (same drag mechanics as action nodes).
- Double-clicking a note enters edit mode with a textarea; pressing Escape or clicking away saves.
- Notes have a delete (×) button to remove them.
- The 📝 button shows a count badge when notes exist.
- Notes persist for the session and participate in canvas panning/zooming.
