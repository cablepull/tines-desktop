# User Story 90: Investigation Artifact Capture

**As a** user preserving a debugging session,  
**I want** key forensic artifacts saved alongside the investigation,  
**so that** evidence survives local cache changes and can be reviewed later.

## Acceptance Criteria
- Saving an investigation can include local artifacts such as screenshot, selected event JSON, selected run JSON, and selected node evidence when available.
- Saved investigations display which artifacts are available.
- Individual artifacts can be downloaded from the investigation panel.
