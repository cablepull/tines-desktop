# Numbered Grid Overlay on Canvas

## Scenario
As a Security Engineer reviewing a large automation graph, I need a visual numbered grid overlay on the canvas that divides the graph into labeled sections. This grid helps me reference specific areas of the graph in discussions and maps directly to the PDF export's stitchable pages.

## Acceptance Criteria
- A "Grid" toggle button in the canvas HUD enables/disables the numbered grid overlay.
- The grid divides the node bounding box into evenly-sized cells.
- Each cell displays its grid number (e.g., "A1", "A2", "B1", etc. or "1", "2", "3").
- The grid is visible in both Visual Canvas and Safety Map modes.
- Grid numbers correspond to the PDF export detail pages.
