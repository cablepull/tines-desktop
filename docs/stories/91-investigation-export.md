# User Story 91: Investigation Export

**As a** user needing to hand off a local investigation,  
**I want** to export it as a portable artifact,  
**so that** I can share a point-in-time summary outside the app.

## Acceptance Criteria
- An investigation can be exported as JSON.
- The export includes status, summary, findings, selected context, and saved artifacts metadata.
- Export is local-only and does not mutate the tenant.
