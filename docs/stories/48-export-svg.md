# Export Graph as SVG

## Scenario
As a Security Engineer, I need to export the current canvas view (Visual or Safety Map) as a clean SVG file for inclusion in documentation, presentations, or audit reports.

## Acceptance Criteria
- An "Export SVG" button triggers a file download of the graph as an `.svg` file.
- The SVG includes all nodes, connecting links, safety badges (in Safety mode), and board notes.
- Works for both Visual Canvas and Safety Map modes.
- The exported SVG is a standalone file with no external dependencies.
