# User Story 93: Investigations Navigation Section

**As a** user working across multiple stories,  
**I want** saved investigations to have their own navigation section,  
**so that** I can browse and reopen them without first opening an individual story.

## Acceptance Criteria
- The main navigation includes an `Investigations` destination.
- The Investigations page lists saved local investigations across stories.
- A user can reopen an investigation from that page and restore its saved context.
