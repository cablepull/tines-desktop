# User Story: In-App Log Console

## Description
**As an** operations analyst diagnosing workflows  
**I want to** pull up a dedicated Log Console directly within the desktop UI  
**So that** I can observe verbose runtime events, raw API responses, and specific error messages without opening Chrome DevTools or checking hidden file logs.

## Acceptance Criteria
- [ ] Implement a global logging mechanism inside the React app.
- [ ] Have the console display a beautiful, readable timeline with precise timestamps.
- [ ] Color-code issues (e.g. Red for failure, Green for successful fetching) to accelerate diagnostics.
- [ ] Keep the console out of the way (e.g., behind a toggle or a slide-up drawer at the bottom of the screen).
