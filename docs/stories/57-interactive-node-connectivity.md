# User Story #57: Interactive Node Connectivity
**As a** Tines Automation Engineer,
**I want to** visually connect nodes by dragging from one to another,
**so that** I can build automation logic topologies directly on the canvas.

### Acceptance Criteria
- Dragging from a node's connection port to another node creates a logic link.
- SVG path updates in real-time during dragging.
- Syncs with `sourceIds` array via Tines API.
