# User Story: Saved Profiles

## Description
**As an** automation engineer handling multiple Tines tenants  
**I want to** save, name, and effortlessly switch between different tenant connection profiles  
**So that** I don't have to re-enter API keys manually every time I want to shift from Development to Production.

## Acceptance Criteria
- [ ] Provide a UI selection menu on the Login Screen displaying previously saved configurations.
- [ ] Allow entering a completely new configuration and naming it.
- [ ] Allow editing or deleting obsolete profiles.
- [ ] Ensure profiles gracefully fail or ask for re-authentication if the API token expires.
